# ONNX-модели

Положите в эту папку один или несколько файлов `*.onnx` и обновите BoxScribe.
Модели появятся в выпадающем списке «ONNX-помощник».

Поддерживаются обычные экспорты детекции YOLOv5 и YOLOv8/YOLO11, а также
end-to-end выходы вида `[x1, y1, x2, y2, score, class]`. Порядок классов модели
должен совпадать с `names` в `data.yaml`.
